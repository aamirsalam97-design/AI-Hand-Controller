# 🖐️ AI Hand Controller

Control your computer mouse using hand gestures with the power of Artificial Intelligence, OpenCV, and MediaPipe.

---

## 📌 Overview

AI Hand Controller is a Python-based Computer Vision project that allows users to control the mouse cursor using their hand.

The project detects hand landmarks in real-time using MediaPipe and maps finger movements to the computer mouse.

---

## ✨ Features

- 🎥 Real-time Hand Detection
- 🖱️ Virtual Mouse Control
- 👆 Index Finger Cursor Movement
- 👍 Thumb + Index Finger Left Click
- ⚡ Smooth Cursor Movement
- 🤖 AI-based Hand Tracking
- 💻 Works with Webcam

---

## 📥 Installation

Clone the repository:

```bash
git clone https://github.com/aamirsalam97-design/AI-Hand-Controller.git
```

Go to project folder:

```bash
cd AI-Hand-Controller
```

Create Virtual Environment:

```bash
python -m venv venv
```

Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Run Project

```bash
python main.py
```
